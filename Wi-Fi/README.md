### Headless 연결 빙법

        boot 폴더에 ssh와 wpa_supplicant.conf 2개의 파일을 복사
        

        wpa_supplicant.conf 파일은 확장자가 .conf인 파일로 확장자를 변경해서는 안됨

        ssid는 wifi의 이름
        psk는 wifi의 비밀번호를 입력한다.

        쌍따옴표 안에다 입력해야함


        ssh 파일과 wpa_supplicant.conf 파일을 라즈베리파이 이미지가 구워진 boot 폴더에 복사하여 사용
